如何在Windows上安装LiiiSTEM ​

第一步：下载安装包 ​

  1. 访问官方下载页面：点击前往
  2. 点击"Windows下载"按钮
  3. 等待下载完成
  4. 如图所示点击 下载 -> 更多操作 -> 保留 -> 显示详细信息 -> 仍然保留

第二步：卸载旧版本 ​

如果您是首次安装LiiiSTEM，可以跳过这一步。如果您之前安装过LiiiSTEM，需要先卸载旧版本。

第三步：运行安装程序 ​

  1. 双击下载的安装包

  2. 由于我们暂未购买Microsoft证书，Microsoft Defender SmartScreen可能会显示安全提示。这是正常现象，我们官网提供的安装包是安全的，请您放心运行。请按以下步骤继续安装：

    • 点击 更多信息
    • 点击 仍要运行

    注：我们正在办理Microsoft证书，此提示将在未来版本中消除。

  3. 按照安装向导的提示进行操作

  4. 选择安装位置（建议保持默认路径）

  5. 等待安装程序完成文件复制

  6. 点击"完成"按钮结束安装

第四步：验证安装 ​

  1. 从开始菜单或桌面快捷方式启动LiiiSTEM
  2. 检查程序是否能正常打开
  3. 测试基本功能是否正常

您可能会遇到的问题 ​

vcruntime140.dll 缺失 ​

vcruntime140.dll在Windows操作系统中扮演着至关重要的角色，它是Microsoft Visual C++ Redistributable Package的一部分。具体作用包括：

运行时支持：vcruntime140.dll是一个动态链接库（DLL）文件，它为使用Visual C++编译器开发的应用程序和游戏提供必要的运行时环境。

库函数：该文件包含了大量应用程序运行时需要调用的库函数，这些函数是实现C++标准库、异常处理机制、RTTI（运行时类型信息）、线程支持等功能的基础。

兼容性：vcruntime140.dll与多个版本的Windows操作系统兼容，确保了不同系统环境下基于Visual C++构建的软件能够正常工作。

应用依赖：许多软件和游戏在安装和运行过程中必须依赖于这个特定版本的vcruntime140.dll文件，如果缺失或损坏，则可能导致相关程序无法启动或者运行中出错。

我们在安装包中已经内置了Microsoft Visual C++ Redistributable Package，但是您可能需要使用管理员权限安装才可以成功。

注意事项 ​

  • 安装过程中请保持电脑开机和网络连接稳定
  • 如遇到安装问题，请查看常见问题解答或联系技术支持团队

如需技术支持，请联系我们的技术支持团队获取帮助。
